# API Perfis de Investimento

API RESTful para simulação e recomendação de investimentos baseados no perfil de risco do cliente.

## Funcionalidades Principais

-  **Autenticação Segura** com JWT
- **Análise de Perfil** de investidor
- **Simulação de Investimentos** com cálculos de rendimentos
- **Recomendações Personalizadas** de produtos
- **Monitoramento** com telemetria
- **Histórico** de simulações

## Começando Rápido

### Via Docker (Recomendado):
```bash
# Clone o repositório
git clone [url-do-repositorio]

# Execute com Docker na raiz do projeto
docker compose up --build

# Acesse a documentação interativa
